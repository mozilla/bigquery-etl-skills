# BigQuery ETL Skills

A comprehensive Claude Code plugin that accelerates development workflows for Mozilla's [bigquery-etl](https://github.com/mozilla/bigquery-etl) repository. This plugin provides:

- **Specialized skills** for writing SQL queries, generating unit tests, managing metadata and schemas, configuring data quality monitoring, and gathering requirements
- **Autonomous agents** that coordinate multiple skills to build complete end-to-end workflows.

All following Mozilla's conventions and best practices.
