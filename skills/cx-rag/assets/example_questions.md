# Example Questions

Sample questions this skill handles well. Use these to set user expectations or to test retrieval quality.

## Sentiment & Mood
- "What are users saying about Firefox password manager?"
- "What is the overall sentiment toward Firefox this month?"
- "Which Firefox features are users most frustrated with?"
- "Are users happy with the new Firefox home screen?"
- "How do users feel about Firefox's memory usage?"

## Topic Discovery
- "What are the most common support topics for Firefox on Android?"
- "What categories of issues do users report after major Firefox updates?"
- "What are users asking about regarding Firefox privacy settings?"
- "What sync-related problems are users reporting?"

## Pain Point Analysis
- "What are the top user pain points in SUMO this week?"
- "What login or account issues come up most frequently?"
- "What are users complaining about with Firefox sync?"
- "What accessibility issues have users raised?"

## Summary Requests
- "Give me a summary of user feedback about Firefox for iOS."
- "Summarize the main themes in SUMO for the Search & Navigation category."
- "What are the most upvoted complaints about tab management?"

## Tips for Better Results
- Be specific about the feature or area (e.g. "Firefox sync bookmarks" vs. "Firefox sync")
- Include a time signal if relevant (e.g. "after the latest update", "this month")
- Use `--top-k 8` for broad topic coverage, default `5` for focused answers
